# Neworbit-eslint-config

ESLint configuration for NewOrbit

## Installation

`npm install neworbit-eslint-config --save-dev`

## Setup

Add `.eslintrc.json` file with the following contents:

```json
{
    "extends": [
        "neworbit-eslint-config"
    ]
}
```
